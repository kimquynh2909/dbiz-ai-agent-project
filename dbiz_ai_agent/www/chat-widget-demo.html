Quick Start

1. Include the SDK

Add the DBiz AI Chat Widget script to your HTML:

<script src="https://your-domain.com/sdk/dbiz-ai-chat-widget.js"></script>

2. Create a Container

Add a container element where you want the chat widget to appear:

<div id="my-chat-widget" data-dbiz-ai-chat></div>

3. Initialize with JavaScript (Optional)

For more control, initialize the widget programmatically:

const chatWidget = new DbizAiChatWidget({
    containerId: 'my-chat-widget',
    baseUrl: 'https://your-api-domain.com',
    apiKey: 'your-api-key', // Optional
    theme: {
        primaryColor: '#3B82F6',
        backgroundColor: '#FFFFFF',
        borderRadius: '12px'
    },
    features: {
        showHeader: true,
        showQuickQuestions: true,
        showTypingIndicator: true
    },
    callbacks: {
        onMessageSent: (message) => console.log('Message sent:', message),
        onMessageReceived: (message) => console.log('Message received:', message),
        onConversationCreated: (id) => console.log('Conversation created:', id),
        onError: (error) => console.error('Error:', error)
    }
});

Configuration Options

Data Attributes

Configure the widget using HTML data attributes:

<div id="my-chat" 
     data-dbiz-ai-chat
     data-base-url="https://your-api-domain.com"
     data-api-key="your-api-key"
     data-conversation-id="existing-conversation-id"
     data-theme='{"primaryColor": "#3B82F6"}'
     data-features='{"showHeader": true}'>
</div>

Theme Customization

  • primaryColor: Main accent color (default: #3B82F6)
  • backgroundColor: Widget background color (default: #FFFFFF)
  • textColor: Main text color (default: #1F2937)
  • borderRadius: Widget border radius (default: 12px)

Feature Options

  • showHeader: Show/hide chat header (default: true)
  • showQuickQuestions: Show/hide quick question chips (default: true)
  • showTypingIndicator: Show typing animation (default: true)
  • showMessageTimestamps: Show message timestamps (default: true)
  • enableFileUpload: Enable file upload feature (default: false)

API Methods

Public Methods

// Send a message programmatically
chatWidget.sendMessage('Hello, AI!');

// Clear all messages
chatWidget.clearMessages();

// Update configuration
chatWidget.updateConfig({
    assistantName: 'New Assistant Name',
    greetingMessage: 'New greeting message'
});

// Get current conversation ID
const conversationId = chatWidget.getConversationId();

// Get all messages
const messages = chatWidget.getMessages();

// Check WebSocket connection
const isConnected = chatWidget.isWebSocketConnected();

// Destroy the widget
chatWidget.destroy();

Event Callbacks

Available Callbacks

  • onReady: Called when widget is initialized
  • onMessageSent: Called when user sends a message
  • onMessageReceived: Called when AI responds
  • onConversationCreated: Called when new conversation starts
  • onError: Called when an error occurs
const chatWidget = new DbizAiChatWidget({
    // ... other options
    callbacks: {
        onReady: () => {
            console.log('Chat widget is ready!');
        },
        onMessageSent: (message) => {
            console.log('User sent:', message.content);
            // Track analytics, etc.
        },
        onMessageReceived: (message) => {
            console.log('AI responded:', message.content);
            // Update UI, etc.
        },
        onConversationCreated: (conversationId) => {
            console.log('New conversation:', conversationId);
            // Save conversation ID, etc.
        },
        onError: (error) => {
            console.error('Chat error:', error);
            // Show error notification, etc.
        }
    }
});

Integration Examples

React Integration

import { useEffect, useRef } from 'react';

function ChatWidget({ baseUrl, apiKey }) {
    const chatRef = useRef(null);
    
    useEffect(() => {
        if (chatRef.current) {
            const widget = new window.DbizAiChatWidget({
                containerId: 'react-chat',
                baseUrl,
                apiKey,
                callbacks: {
                    onMessageReceived: (message) => {
                        // Handle AI response in React
                        console.log('AI response:', message);
                    }
                }
            });
            
            return () => widget.destroy();
        }
    }, [baseUrl, apiKey]);
    
    return <div id="react-chat" ref={chatRef} />;
}

Vue.js Integration

<template>
    <div id="vue-chat" ref="chatContainer"></div>
</template>

<script>
export default {
    props: ['baseUrl', 'apiKey'],
    mounted() {
        this.chatWidget = new window.DbizAiChatWidget({
            containerId: 'vue-chat',
            baseUrl: this.baseUrl,
            apiKey: this.apiKey,
            callbacks: {
                onMessageReceived: (message) => {
                    this.$emit('message-received', message);
                }
            }
        });
    },
    beforeUnmount() {
        if (this.chatWidget) {
            this.chatWidget.destroy();
        }
    }
};
</script>

Mobile Responsiveness

The chat widget is fully responsive and works great on mobile devices. It automatically adjusts its layout for different screen sizes and provides touch-friendly interactions.

/* Mobile-specific styling */
@media (max-width: 768px) {
    .dbiz-chat-widget {
        height: 100vh; /* Full screen on mobile */
        border-radius: 0;
    }
    
    .dbiz-message {
        max-width: 90%; /* Larger messages on mobile */
    }
}
